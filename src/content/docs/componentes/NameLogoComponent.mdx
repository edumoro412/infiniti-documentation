# NameLogoComponent

Este componente muestra el logo y el nombre de la aplicación, enlazando a la página principal. Utiliza un ícono y un título estilizados.

## Funcionamiento

- Muestra un ícono (infinito) y el texto "INFINITI".
- El logo es un enlace que redirige a la página principal (`/`).
- Utiliza el componente `Icon` para mostrar el ícono.

## Ejemplo de uso en MDX

```mdx
import NameLogoComponent from "./NameLogoComponent.vue";

<NameLogoComponent />
```

## Props

No recibe props externas. El contenido y estilos están definidos en el template.

## Personalización

Puedes cambiar el ícono o el texto modificando el template del componente.

## Estilos

Incluye estilos SCSS para el logo, el enlace y la disposición de los elementos.
