# Introducción

Bienvenido a la documentación de **INFINITI**.

Este proyecto es una aplicación web construida con Nuxt 3 y Vue 3, diseñada para mostrar noticias, información meteorológica y ofrecer una experiencia personalizada con soporte para múltiples idiomas y temas (oscuro/claro).

## Características principales

- Visualización de noticias por categorías.
- Búsqueda de noticias.
- Información meteorológica integrada.
- Soporte multilenguaje (i18n).
- Tema oscuro y claro con persistencia.
- Animaciones Lottie.
- Componentes reutilizables y bien estructurados.

Esta documentación te guiará a través de la instalación, configuración, estructura y uso del proyecto, así como de sus componentes y utilidades clave.
